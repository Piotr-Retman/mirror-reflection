How to play?

The concept

This is pretty easy idea. Player have the draft on the left side of the playground. The object marked by yellow squares is shape which has to be imitate on the red squares. So your task is make the "reflection" of the shape from green area squares. Seems easy...but it is not so simple always :)

Gameplay

Different game modes

Different configurations can give you different points during the play

Infinite chances

This game mode let you click as much as you can to reflect the shape.

Time counting

This game mode as you probably guess will count time during the level.Maximum time to solve the level is 2 minutes.

Does the game modes can be mixed?

This game was primarily created for fun ... so yes! of course you can mix game modes. At any moment if you think the mode is too annoying or too difficult you can reduce the difficulty of the game. Remember, however, that the simpler the game mode the less points you get! If you want to be really high in the rankings. You must be very fast. Faster than a typical person :)

Bonus

Average clicker click more or less one per 215ms. So if you are faster than that you can get extra points! Points coefficient will be highest and you will be on top of the ranks!
